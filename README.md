Тестовое задание Sirius Future.

Используемые технологии:
1) Nextjs.
2) Styled-components.
3) Библиотека emothion.sh.
4) Storybook для создания библиотеки компонентов.

Для запуска проекта:
1) Клонируем репозиторий.
2) Устанавливаем зависимости.

*npm install*

3) Запускаем storybook.

*npm run storybook*
